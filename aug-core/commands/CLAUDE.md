# Commands

## Purpose

Slash commands for aug-core plugin. Focus on AI-enhancement, session management, automation, and code pattern analysis.

## Key Components

**Session Management:**
- `notetoself.md` - Save session context to /tmp
- `futureme.md` - Resume from saved context

**Prompt Crafting:**
- `hemingway.md` - Ruthlessly concise rewriting

**Automation:**
- `automate.md` - Autonomous command execution with user-standin agent

**Workflows:**
- `workflow-run.md` - Execute multi-command workflows
- `workflow-status.md` - Check workflow progress

**Code Patterns:**
- `learn.md` - Analyze codebase conventions
- `suggest.md` - Compare file to conventions
- `patterns.md` - Show all detected patterns

**Team Variants:**
- `create-variant.md` - Generate team-specific workflow files

## Usage

Commands invoked via `/command-name` in Claude Code. See individual files for syntax and options.
