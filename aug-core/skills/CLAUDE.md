# Skills

Reusable skill definitions for aug-core plugin. Focus on prompt optimization, workflow design, and code pattern detection.

- `workflow-design/` - Multi-command workflow patterns

## Usage

Skills invoked via Skill tool. Referenced by commands and other skills for consistent behavior.
