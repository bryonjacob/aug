# Aug-Util Plugin

Session management and working context utilities.

## Installation

```bash
/plugin install aug-util@aug
```

## Commands

- `/notetoself` - Save current working context to /tmp
- `/futureme` - Resume from previously saved context

## Use Cases

- Context usage approaching limits
- Need to switch projects temporarily
- End of work session with incomplete tasks
- Complex multi-step work requiring multiple sessions
