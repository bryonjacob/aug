# Commands

## Purpose

Slash commands for aug-just plugin. Justfile adoption, assessment, and upgrade workflows.

## Key Components

- `just-init.md` - Install baseline justfile for stack
- `just-assess.md` - Assess current justfile maturity
- `just-refactor.md` - Make justfile conform to standard
- `just-upgrade.md` - Add next maturity level or pattern

## Usage

Commands invoked via `/command-name` in Claude Code. Guide projects through justfile maturity progression.
