# Skills

## Purpose

Reusable skill definitions for aug-just plugin. Justfile patterns organized by maturity level.

## Key Components

**Core Patterns:**
- `justfile-interface/` - Level 0 - Standard command interface
- `justfile-quality-patterns/` - Level 1 - Test watch, complexity, duplicates
- `justfile-security-patterns/` - Level 2 - Security scanning
- `justfile-advanced-patterns/` - Level 3 - Performance, monitoring
- `justfile-polyglot-patterns/` - Multi-language orchestration

**Meta:**
- `justfile-maturity-model/` - 5-level progression framework

## Usage

Skills invoked via Skill tool. Each pattern skill defines commands, when to add them, and implementation examples across stacks.
