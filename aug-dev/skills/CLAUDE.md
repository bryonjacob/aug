# Skills

## Purpose

Reusable skill definitions for aug-dev plugin. Core development workflows, stack configurations, and quality standards.

## Key Components

**Development Skills:**
- `software-architecture/` - Epic planning and design
- `software-development/` - Implementation following specs
- `software-debugging/` - Systematic bug finding
- `software-quality/` - Test coverage strategy

**Stack Configuration:**
- `configuring-javascript-stack/` - JS/TS toolchain
- `configuring-python-stack/` - Python toolchain
- `configuring-java-stack/` - Java toolchain
- `configuring-polyglot-stack/` - Multi-language orchestration
- `justfile-standard-interface/` - Standard justfile commands

**Development Standards:**
- `installing-git-hooks/` - Pre-commit/pre-push hooks
- `configuring-github-actions/` - CI/CD workflows
- `documenting-with-claude-md/` - CLAUDE.md hierarchy
- `documenting-with-mkdocs/` - MkDocs documentation
- `documenting-with-audit/` - Automated doc auditing

**Development Workflows:**
- `executing-development-issues/` - Complete issue lifecycle
- `refactoring/` - Refactoring with coverage gates
- `self-reviewing-code/` - Pre-PR checklist
- `working-in-git-worktrees/` - Parallel work isolation
- `creating-agents/` - Agent definition patterns
- `development-stack-standards/` - Stack maturity models

## Usage

Skills invoked via Skill tool. Referenced by commands and other skills for consistent behavior.
