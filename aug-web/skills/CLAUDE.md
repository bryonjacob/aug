# Skills

## Purpose

Reusable skill definitions for aug-web plugin. Web development patterns and integrations.

## Key Components

- `building-static-sites/` - Static site generation patterns
- `building-with-nextjs/` - Next.js application patterns
- `integrating-formspree-forms/` - Formspree form integration
- `styling-with-tailwind-cva/` - Tailwind + CVA styling patterns

## Usage

Skills invoked via Skill tool. Provide web-specific patterns extending JavaScript stack configuration.
